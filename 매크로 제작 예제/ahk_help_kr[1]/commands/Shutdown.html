Shutdown

시스템을 종료, 재기동, 로그 오프 한다

Shutdown, Code

Parameters

인수명설명
Code 이하의 것의 화
0
로그 오프
1
슛다운
2
재기동
4
강제적
8
실행 후 전원을 끈다

Remarks

OnExit써브루틴을 설정해 두어, 「A_ExitReason」변수를 조사하면, 유저가 슛다운이나 로그 오프를 실시하려고 한 것을 알 수 있다.

Related

Run, ExitApp, OnExit

Example(s)

; Force a reboot (reboot + force = 2 + 4 = 6):
Shutdown, 6
